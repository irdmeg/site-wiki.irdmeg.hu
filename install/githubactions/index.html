    Github actions

    https://docs.docker.com/ci-cd/github-actions/

    • Hozz létre github repót, vagy abban a repoban adj hozzá workflows-t amelyikben használni szeretnéd a github-actions-t
    • A repó settings fülének a a secret menűpontjában adjuk hozzá a Repository secrets-eknél adjuk hozzá a doker hubunkat.
    • Docker hubon ezt a tokent az account settingsnél tudjuk beállítani. > Security és új token létre hozzása, nevet adjunk neki és létre hozzuk.
    • Új workflow-ot hozzunk létre a repó actions menűpontjában.
