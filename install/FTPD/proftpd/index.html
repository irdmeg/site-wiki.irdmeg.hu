    Default config

    1
    nano /etc/proftpd/proftpd.conf

    A fenti beállítást megváltoztathatja a követelményei szerint, az alábbiak szerint:

    • Kiszolgálónév: Megváltoztathatja alapértelmezett kiszolgálónévként.
    • UseIPV6: Kikapcsolhatja.
    • DefaultRoot: A sor visszavonása nélkül korlátozhatja a felhasználókat az otthoni mappákkal.
    • Port: Megadhatja a saját portját annak megváltoztatásával.
    • SystemLog: A naplófájl alapértelmezett helye. Megváltoztathatja azt az Ön igényei szerint.

#</IfModule>
